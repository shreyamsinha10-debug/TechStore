HTML Basics

HTML is a markup language used to structure content on web pages using tags like <html>, <head>, and <body>.

<!DOCTYPE html>
<html>
  <head>
    <title>My Page</title>
  </head>
  <body>
    <h1>Hello World</h1>
  </body>
</html>

HTML Elements

An element is everything from the start tag to the end tag, for example <p>...</p>.

<p>This is a paragraph element.</p>

HTML Attributes

Attributes provide extra information in the start tag, such as href, src, alt, id, and class.

<a href="https://www.example.com">Visit Example</a>

HTML Headings

Headings go from <h1> (most important) to <h6> (least important).

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>

HTML Paragraphs

Paragraphs are defined with the <p> tag and are block‑level elements.

<p>This is a paragraph of text.</p>

HTML Inline Styles

The style attribute can apply CSS directly to an element, for quick demonstrations.

<p style="color: blue; font-size: 20px;">Styled text</p>

HTML Text Formatting

Common tags include <b>, <strong>, <i>, <em>, <small>, <mark>, and others.

<p>This is <strong>important</strong> and <em>emphasized</em> text.</p>

HTML Comments

Comments are ignored by the browser and start with <!-- and end with -->.

<!-- This is a comment -->

HTML Colors

Colors can be set with names, hex codes, RGB, or HSL in CSS.

<p style="color: red;">Named color</p>
<p style="color: #00ff00;">Hex color</p>

HTML Links

Links are created with the <a> tag and the href attribute.

<a href="https://www.w3schools.com">Go to W3Schools</a>

HTML Images

Images use the <img> tag with src and alt attributes and do not need a closing tag.

<img src="logo.png" alt="Site logo" width="100" height="100">

HTML Tables

Tables use <table>, rows with <tr>, header cells with <th>, and data cells with <td>. [web:11]

<table border="1">
  <tr>
    <th>Name</th>
    <th>City</th>
  </tr>
  <tr>
    <td>Alex</td>
    <td>Bangalore</td>
  </tr>
</table>

HTML Lists

Unordered lists use <ul> and ordered lists use <ol>, both containing <li> items.

<ul>
  <li>Tea</li>
  <li>Coffee</li>
</ul>

Block vs Inline

Block elements (like <div>, <p>) start on a new line and take full width, while inline elements (like <span>, <a>) stay in the flow of text. [web:10]

Block 1
Block 2
Inline A Inline B

Class and Id

class can be reused on multiple elements, while id is meant to be unique in a page.

<p class="note">Reusable style</p>
<p id="main-title">Unique element</p>

HTML Iframes

Iframes embed another page inside the current page using the <iframe> tag.

<iframe src="https://www.example.com" width="300" height="200"></iframe>

HTML Head

The <head> section contains metadata, title, links to CSS, scripts, and other resources. [web:10]

<head>
  <meta charset="UTF-8">
  <title>Page Title</title>
  <link rel="stylesheet" href="style.css">
</head>

HTML Forms

Forms collect user input using the <form> tag and controls like <input>, <textarea>, and <select>. [web:10]

<form action="/submit" method="post">
  <label>Name:
    <input type="text" name="username">
  </label>
  <input type="submit" value="Send">
</form>

CSS Introduction

CSS controls the presentation of HTML elements and can be added inline, in the <head>, or in external stylesheets. [web:24]

/* External or internal CSS */
p {
  color: darkblue;
}

CSS Selectors

Selectors choose elements to style: type selectors (p), class selectors (.note), id selectors (#main), and universal selector (*). [web:24][web:17]

p { color: green; }
.note { background: #ffffcc; }
#main { border: 1px solid #000; }
* { margin: 0; padding: 0; }

CSS Colors

CSS sets colors for text, backgrounds, borders, and more using names, hex, RGB, or HSL. [web:24]

h1 { color: #ff6600; }
div { background-color: rgb(240, 240, 240); }

CSS Backgrounds

Background properties include background-color, background-image, background-repeat, and background-size. [web:24]

body {
  background-color: #fafafa;
}
.hero {
  background-image: url("banner.jpg");
  background-size: cover;
}

CSS Borders

Borders are controlled with border-width, border-style, and border-color or the shorthand border. [web:24]

.box {  
  border: 2px solid #333;
  border-radius: 5px;
}

CSS Margins

Margins create space outside elements using margin-top, margin-right, margin-bottom, and margin-left, or the shorthand margin. [web:24]

.container {
  margin: 20px; /* all sides */
  margin: 10px 15px; /* vertical | horizontal */                
    margin: 5px 10px 15px 20px; /* top | right | bottom | left */   
}
